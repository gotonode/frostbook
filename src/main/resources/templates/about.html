About

Frostbook (hereafter refererred to as the "app" or "service") is a social media platform, made as a student project. Read this document to learn more about the technical details of this project.

Database

During development, H2 database engine is used with a flatfile database in cleartext form. Once the app is pushed to Heroku, Heroku's own PostgreSQL is used for data persistence accross server restarts.
